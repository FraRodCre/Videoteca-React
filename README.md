# Vieteca-React

Url de acceso:
[http://localhost:3000/](http://localhost:3000/)

**Nota: Es necesario añadir en tu API KEY en el fichero VideotacaApps.js en la constante const API_KEY = 'your api key'**

## Instalación

Instalar depencencias:

```shell
npm install
```

## Arrancar la App

### Modo desarrollo:

```shell
npm start
```

### Modo producción:

```shell
npm run build
```
